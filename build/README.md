# Point Cloud Scripts

## Core Scripts

### main.sh

**Usage:**

```bash
bash main.sh [input_cloud] [config_file]
```

### visualize.sh

**Usage:**

```bash
bash visualize.sh [input_cloud] [config_file] [color_space] [overlay=True] [gray=True]
```

## Other Scripts

### view_cloud.py

**Usage:**

```bash
python3 view_cloud.py [path_to_input_cloud]
```

### overlay_cloud.py

**Usage:**

```bash
python3 overlay_cloud.py [input_cloud] [segmented_cloud] [overlay=True] [gray=True]
```
